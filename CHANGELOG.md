# Changelog

All notable changes to BilevelPy will be documented in this file.

The project follows [Semantic Versioning](https://semver.org/).

## [0.1.0] — 2026-07-23

Initial public alpha release.

### Added

- Extensible framework for building, solving, and evaluating Gurobi-based optimization models
- Composable variables and constraints with dependency validation
- Structured data layer with loaders and preprocessing pipelines
- Built-in support for hub-location models and CAB benchmark datasets
- Configurable Gurobi execution and structured solution extraction
- Benchmarking utilities with structured result logging
- MkDocs documentation with tutorials, guides, and an API reference
- Automated test suite with 175 tests
- Continuous integration for Python 3.11, 3.12, 3.13, and 3.14
- Poetry-based packaging and wheel validation
- GitHub Pages documentation deployment
